= 001 - Eks standard with IRSA authentication

[NOTE]
Both the cluster and the node group currently deploy successfully as of 20260129 @ 1230. I have also been successful in deploying ArgoCD to this cluster

This project should only be executed using the CI environment and will build a new Aws EKS cluster with correctly attached roles for access from your SSO role, the CI role, or the generated EKS role.

== How do deploy?

* Make sure the `account-resources` project has been executed
* Make sure the `ci-vpc` project has been run from ci
* Run this project from ci
